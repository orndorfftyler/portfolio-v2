Portfolio

BookTrove

BookTrove is an app which lets you leave reviews for your favorite books and mark other reviews as useful. The search is powered by Google Books. It was built using React, Node.js, PostgreSQL, Express, and Knex.

Event Weather App

The event weather app provides a weather forecast for a given event. The Ticketmaster API is used to retrieve event information and the Open Weather API is used for the weather forecasts. It was built using HTML, CSS, JavaScript, and jQuery.

Aquatic Answers

Aquatic Answers lets users ask and answer questions related to aquariums. Anyone can view questions, but an account is required to post. It was built using React, Node.js, PostgreSQL, Express, and Knex.
